SabiGuy Mobile is the mobile application for the SabiGuy platform, a marketplace that connects customers with trusted service providers for everyday tasks and professional services.

The app allows users to discover services, hire providers, chat in real-time, manage bookings, and handle secure payments directly from their mobile devices.

Features
User Features
Browse and search for service providers
Create and manage bookings
Real-time chat with providers
Secure wallet payments
Booking status tracking
Notifications for booking updates
Rate and review providers

Provider Features
Receive job requests
Accept or decline bookings
Real-time chat with customers
Manage ongoing jobs
Track earnings and payments
Receive booking alerts
Platform Features
In-app messaging system
Wallet and payment system
Booking lifecycle management
Push notifications
Secure authentication
